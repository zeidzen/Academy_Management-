USNews is a Multipurpose News, Magazine and Blog HTML5 Template Specially designed for all kinds of News, Magazine and Reviews Websites. It's an HTML5 template based on latest stable bootstrap 3.3.7. Anyone can easily update/edit this template to follow our Well Sorted Documentation.
